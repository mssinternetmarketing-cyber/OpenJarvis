# Kevin's Update Recommendations

Drop any file here that you want the Update Agent to evaluate.
The agent will:
  1. Assess validity and safety
  2. Improve in sandbox if needed
  3. Version and log everything
  4. Send approved files to SendToOpenJarvisUpdaterFiles/ for your review
  5. Alert you about any risky changes requiring human decision

Commands:
  SCAN UPDATES  — run the update agent now
  SHOW UPDATES  — see what's queued for review
  APPROVE UPDATE <filename>  — approve and queue for deployment
  REJECT UPDATE <filename>   — reject and log
